# Flask Project 🚀

Учебный проект на Flask с набором практических заданий для изучения веб-разработки.

## 📝 Описание

Это Flask веб-приложение, созданное во время обучения на курсах. Проект содержит несколько функциональных модулей, каждый из которых реализует отдельное учебное задание.

## ✨ Функции

- **Главная страница** — Welcome страница приложения
- **Регистрация** — Регистрация пользователя с проверкой возраста (18+)
- **Todo список** — Управление списком задач (добавление новых задач)
- **Калькулятор ИМТ** — Расчёт индекса массы тела по формуле: вес / (рост²)
- **Регистрация аккаунта** — Создание аккаунта с проверкой совпадения паролей
- **Админ-панель** — Вход в админ-панель (демо пароль: `1234`)
- **Обратная связь** — Форма для отправки сообщений обратной связи

## 📋 Требования

- Python 3.7+
- Flask

## 🛠️ Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/baysangur095/flask-project.git
cd flask-project
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустите приложение

```bash
python main.py
```

Приложение запустится на `http://localhost:5000`

Это учебный проект. Некоторые функции (например, сохранение данных) реализованы в памяти приложения и не сохраняются между перезагрузками. Для продакшена требуется интеграция с базой данных.
